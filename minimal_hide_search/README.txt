This plugin contains the most simple code to hide the search box from site visitors who are not logged-in.


Install instructions:

1. Copy the plugin folder in your mod directory,
2. Enable the plugin in the admin section of your site,
3. Done.


Changelog:

1.9.0:

- Updated for Elgg 1.9.
(If you use the Aalborg theme included in Elgg 1.9 you won't need this plugin as the theme already hides the search box from logged-out visitors.)


1.0:

- First and most likely last release.
